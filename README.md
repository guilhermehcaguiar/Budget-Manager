# 💰 Budget Manager

Dashboard financeiro completo com React + Node.js. Visualize receitas, despesas e gráficos automáticos por categoria.

---

## 🧩 Funcionalidades

- ✅ Cadastro de receitas e despesas por categoria
- ✅ Filtro por mês e ano
- ✅ Gerenciamento de categorias
- ✅ Gráfico de barras: receitas vs despesas por categoria
- ✅ Gráfico de pizza/donut: participação de cada categoria
- ✅ Breakdown detalhado por categoria com barra de progresso
- ✅ Taxa de poupança calculada automaticamente
- ✅ Exclusão de lançamentos
- ✅ 4 abas: Visão Geral, Gráficos, Categorias, Lançamentos

---

## 🚀 Como rodar

### 1. Instalar dependências
```bash
npm install
```

### 2. Rodar em modo desenvolvimento (servidor + cliente juntos)
```bash
npm run dev
```

Isso inicia:
- **Frontend React** → http://localhost:3000
- **Backend Node/Express** → http://localhost:3001

### 3. Build para produção
```bash
npm run build
npm start
```
